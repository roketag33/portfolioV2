# Gravity Physics Playground

A 3D physics sandbox where you can spawn objects and manipulate gravity in real-time.

## Features
- Variable gravity controls (X, Y, Z axes)
- Object spawning (cubes, spheres)
- Interactive physics simulation

## Tech Stack
- **3D Engine**: Three.js / React Three Fiber
- **Physics**: Cannon.js / React Three Cannon
- **UI**: React / Tailwind CSS
